RowDB

Personal Data Table Manager — Lightweight, interactive C++ database system for your console.

About RowDB

RowDB is a simple, open-source C++ application for managing personal data tables. It features an interactive command-line interface, ASCII table display with row numbers, and supports saving/loading tables in a human-readable format.
